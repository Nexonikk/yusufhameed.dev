About me

The engineer behind
the backend

I don't just build features. I build systems that hold up under load.

I'm a full stack developer with 1+ year of experience building scalable web applications, backend APIs, and distributed systems using Node.js, TypeScript, Next.js, and Java Spring Boot. I've worked hands-on with AWS, Docker, Redis, RabbitMQ, and CI/CD pipelines in production environments, with a strong foundation in REST API design, microservices architecture, and SOLID principles.

Currently, I contribute to AI evaluation and benchmarking infrastructure at Xelron AI, designing standalone backend and system-level tasks used to train and evaluate large language models within the Snorkel AI Terminus / Terminal-Bench framework - building Docker-based Linux environments with pinned dependencies for reproducible, deterministic task runs.

I bring an above-and-beyond approach to my work. My goals aren't limited by familiarity or job descriptions - I'm a problem-solving fanatic with a relentless devotion to efficiency.

Previously, I worked as a Full Stack Developer at Talecto, architecting type-safe REST APIs with Node.js and Express, integrating AWS S3 for large-scale data storage, and improving authentication system performance with Redis caching inside a Dockerized environment. For automation and API integrations, I work with Claude Code via terminal at an advanced level - building CLI-based workflows that didn't exist before.
